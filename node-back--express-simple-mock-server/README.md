### Servidor node para mock simple para desarrollos locales

* Node JS
* Express
* Morgan
* Nodemon

### Instalacion
* instalar modulos
```
    npm i express morgan
    npm i cors
    npm i nodemon -D # para que se instalado en entorno development
```
* Iniciar servidor 
```
    npm run dev
```
